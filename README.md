# Лабораторна робота №3

## 📚 Тема
Реалізація задач з використанням графів

## 🧩 Зміст

У лабораторній роботі реалізовані наступні задачі:

1. **Пошук у ширину (BFS)**  
   - Знайти найкоротший шлях між двома вершинами в орієнтованому графі.

2. **Граф — список суміжності**  
   - Обробка графа заданого списком суміжності.

3. **Граф — матриця суміжності**  
   - Підрахунок кількості ребер у неорієнтованому графі.

4. **Зважений граф — список ребер**  
   - Обчислення загальної ваги ребер та знаходження ребра з мінімальною і максимальною вагою.

5. **Перевірка орієнтованості графів**  
   - Визначення, чи є задані графи орієнтованими.

6. **Транзитивне замикання графа**  
   - Побудова транзитивного замикання для графа.
